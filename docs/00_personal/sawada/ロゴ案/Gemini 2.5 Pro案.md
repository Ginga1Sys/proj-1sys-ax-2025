# システム名称・ロゴ案 (Gemini 2.5 Pro案)

## 1. AI-Hub (アイハブ)

### コンセプト
「AIに関する情報や人が集まる中心地」を表現。

### ロゴ案
- **デザイン**:
    - 中央に「AI」の文字を配置し、そこから複数の線が放射状に伸びて、周囲の点（人や情報を象徴）と繋がっているデザイン。
    - 六角形のハニカム構造の中心に「AI」を置き、周囲のセルにナレッジやアイデアを意味するアイコンを配置する。
- **カラー**: 信頼感のある青を基調とし、繋がりや活性化を表現するオレンジや緑をアクセントカラーに使用。

## 2. Axon (アクソン)

### コンセプト
「社内にAIナレッジを隅々まで行き渡らせる」という情報の伝達を表現。

### ロゴ案
- **デザイン**:
    - 神経細胞（ニューロン）の軸索（Axon）が枝分かれしていく様子をモチーフにする。
    - 「Axon」の"X"の文字を、情報が交差・伝達するポイントとして強調したデザイン。
- **カラー**: 知性を感じさせる濃紺や紫をメインに、情報の流れを光のライン（水色や白）で表現。

## 3. Synapse (シナプス)

### コンセプト
「人と人、情報と情報をつなぎ、新たなアイデアが生まれる場所」を表現。

### ロゴ案
- **デザイン**:
    - 2つのニューロンがシナプスで接続する部分を抽象的にデザイン化する。
    - 「Synapse」の"S"の文字を、2つの要素が繋がるような形状にする。接続部分が光っているようなエフェクトを加える。
- **カラー**: 創造性やひらめきを刺激するグラデーションカラー（例：青から紫、ピンクへ）を使用。

## 4. AI-Pedia (アイペディア)

### コンセプト
「AIに関する社内版の百科事典」というナレッジの集積を表現。

### ロゴ案
- **デザイン**:
    - 開かれた本や、パズルのピースが組み合わさってできた地球儀のアイコンに「AI」の文字を組み合わせる。
    - 虫眼鏡のアイコンと「AI」を組み合わせ、探している情報が見つかるイメージを表現。
- **カラー**: 知識や信頼性を象徴する緑や青を基調とする。

## 5. AI-Compass (アイコンパス)

### コンセプト
「AI活用の方向性を示す羅針盤」を表現。

### ロゴ案
- **デザイン**:
    - 羅針盤（コンパス）をモチーフにし、針の先端が「AI」の文字を指しているデザイン。
    - 「AI-Compass」の"C"の文字をコンパスの形にする。
- **カラー**: 信頼と先進性を表すブルーシルバーや、ゴールドをアクセントに使用し、高級感を演出。
