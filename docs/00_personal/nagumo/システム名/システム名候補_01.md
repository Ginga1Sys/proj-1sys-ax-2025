# システム名候補チェックテンプレート

以下は上位3候補（`AIナレッジハブ`、`シナプスAI`、`NaviAI`）向けの評価チェックテンプレートです。
各項目について「評価（1-5）」と「コメント／確認結果」を記入してください。

---

**使い方**
- 各候補ごとに同じテンプレートをコピーして評価してください。
- 評価は1（低）〜5（高）で記入。必要に応じてコメントで根拠を示してください。
- ドメインやSNSの可否、商標リスクは別途実施するチェックのリンクをここに貼るか、下部の備考欄に結果をまとめてください。
