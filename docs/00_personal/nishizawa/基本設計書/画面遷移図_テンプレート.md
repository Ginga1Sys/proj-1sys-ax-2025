# 画面遷移図

## 1. 概要
本ドキュメントは「AI活用ナレッジ共有サイト」の画面遷移を定義する。

## 2. 画面遷移図（Mermaid）

```mermaid
graph LR
    %% 認証系
    START((開始)) --> Login[ログイン画面]
    Login --> Register[会員登録画面]
    Register --> Login

    %% メイン機能
    Login --> Dashboard[ダッシュボード]
    Dashboard --> Search[検索結果一覧]
    Dashboard --> ArticleDetail[記事詳細画面]
    Dashboard --> PostArticle[記事投稿・編集画面]

    %% AI連携
    ArticleDetail --> AI_Summary[AI要約生成]
    PostArticle --> AI_Suggest[AIタグ提案]

    %% ユーザー設定・管理
    Dashboard --> Profile[プロフィール設定]
    Dashboard --> Admin[管理者パネル]

    %% 戻り遷移
    Search --> ArticleDetail
    ArticleDetail --> Dashboard
    PostArticle --> ArticleDetail
    Admin --> Dashboard
```
